📄 PROJECT REPORT
Title: Adaptive Powerlifting Load Calculator Using RPE-Based Estimation
1. Introduction

Strength training, particularly powerlifting, relies heavily on managing training intensity and volume. A widely used method for gauging intensity is the Rate of Perceived Exertion (RPE) scale, which allows lifters to estimate how close they are to muscular failure.

However, many lifters struggle with:

Accurately selecting weights for target RPEs
Adjusting loads dynamically based on performance
Following structured programs like 5/3/1 while adapting to daily readiness

This project aims to solve this problem by building an intelligent system that recommends training weights based on user input and program structure.

2. Problem Statement

Lifters often rely on intuition or static percentage-based programs, which:

Do not adapt to daily performance variations
Lead to undertraining or overtraining
Require experience to execute effectively

There is a need for a simple, adaptive tool that:

Uses RPE to estimate strength
Suggests appropriate weights for future sets
Integrates structured programs like 5/3/1
3. Objective

The objective of this project is to develop a Python-based application that:

Accepts user input (weight, reps, RPE)
Estimates the user’s one-rep max (1RM)
Suggests the appropriate weight for a target set
Supports both manual input and structured program modes
Provides realistic outputs by rounding weights to available gym plates

6. Key Design Decisions
1. Rule-Based Instead of Complex ML

Instead of using heavy machine learning models, a mathematical model was chosen because:

It is interpretable
Works reliably with small data
Is sufficient for the problem scope
2. Streamlit for UI

Chosen because:

Fast to develop
Requires minimal frontend knowledge
Ideal for prototyping ML applications
3. Dropping Video-Based RPE Estimation

Initially considered, but excluded due to:

High complexity
Time constraints
Lack of dataset

It is suggested as future work.

7. Challenges Faced
